# Swift Code Snapshot

Swift 코드를 Xcode 스타일 스냅샷 이미지로 만들어주는 도구입니다.

## 기능

- Swift 구문 강조 (키워드, 타입, 문자열, 숫자, 주석, @속성)
- 특정 줄 하이라이트 (범위 지정 가능: `3,5,8-10`)
- Light / Dark 테마 전환
- 📋 이미지 클립보드 복사 (Cmd+V로 키노트에 바로 붙여넣기)
- 💾 PNG 고해상도(3x) 다운로드
- HTML 소스 복사

## 사용법

1. 왼쪽에 Swift 코드 붙여넣기
2. 하이라이트할 줄 번호 입력 (예: `8-10`)
3. **미리보기 →** 클릭 (또는 `Cmd+Enter`)
4. **📋 이미지 복사** → 키노트/피그마에 `Cmd+V`

## GitHub Pages 배포

1. 이 레포를 GitHub에 push
2. Settings → Pages → Source를 `main` 브랜치, `/ (root)` 선택
3. `https://<username>.github.io/swift-code-snapshot/` 에서 접속
